# Problem 50. Pow(x,n)

Implement pow(x, n), which calculates x raised to the power n (i.e., xn).

## My Solution

```js
//  My Solution
/**
 * @param {number} x
 * @param {number} n
 * @return {number}
 */
var myPow = function (x, n) {
  return Math.pow(x, n);
};
```

It runs great (69.48% runtime, 67.45% memory)
